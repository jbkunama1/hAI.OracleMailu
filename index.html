📬 hAI.OracleMailu

Komplette Anleitung für einen selbst gehosteten Mailserver mit Mailu auf Oracle Cloud Always Free – mit Docker, Portainer, cloudflared und Cloudflare DNS. Inklusive englischer Doku.

🖥️ Oracle A1 Flex ARM 🐳 Docker Compose 🔒 cloudflared Tunnel 🌐 Cloudflare DNS 📨 SMTP / IMAP 🛡️ SPF / DKIM / DMARC ⚡ Always Free
🇩🇪 README (DE) 🇬🇧 README (EN) 📦 GitHub Repo

🚀 Was enthalten ist

  • Oracle A1 Flex VM erstellen (Schritt für Schritt)
  • Out-of-Capacity Problem lösen
  • Docker, Portainer, cloudflared Setup
  • Mailu als Docker Stack deployen
  • Cloudflare DNS mit SPF, DKIM, DMARC
  • Port 25 Freischaltung bei Oracle
  • Zustellbarkeit testen und optimieren

📊 Ressourcennutzung

DienstRAM
Mailu gesamt~1.500 MB
Portainer~100 MB
cloudflared~50 MB
System + Docker~400 MB
Gesamt~2.050 MB ✅

⚠️ Wichtige Hinweise

  • Cloudflare-Proxy beim mail-Record deaktivieren
  • Oracle blockiert Port 25 standardmäßig → Ticket nötig
  • PTR, SPF, DKIM und DMARC für gute Zustellung essenziell
  • A1 Flex: bei Out of Capacity → Pay As You Go Trick

🪜 Setup-Ablauf

1

Anmelden

Oracle Cloud Account, Region prüfen.

2

VM erstellen

A1 Flex ARM, 2 OCPU, 4 GB, Debian/Ubuntu, Public IP, SSH-Key.

3

SSH & System

Verbinden, Updates, Hostname, Basis-Tools.

4

Netzwerk

Oracle Security List + iptables: 7 Ports öffnen.

5

Docker

Docker Engine + Portainer installieren.

6

Tunnel

cloudflared für sichere Admin-Zugänge.

7

Mailu

Konfiguration via setup.mailu.io, Stack starten.

8

DNS & Tests

Cloudflare DNS setzen, Port 25 freischalten, Testmail.
